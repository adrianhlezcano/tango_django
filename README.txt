This is a web project to store urls of different pages by certain category. The main idea of this app is to track your favorites and share with others people. You have to create an account to add/edit pages links or you can only see the links of other users. 
Main Features:
  > Save page links.
  > Search pages links using the Bing Search Engine.
  > Register as a user - Access to your profile.
  > Look for websites by categories.
  > Track the number of views of each page.
  > Add a "like(s)" to categories or pages.

This web app is the result of the tutorial: www.tangowithdjango.com
